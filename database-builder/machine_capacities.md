# Machine Ingredient Capacities

Realistic maximum hopper/container capacity per ingredient, per machine model.
Sized so that a full refill lasts **roughly 5 days** at observed average usage,
rounded up to practical container sizes.

Assume water is unlimited

Locations: Lviv (UA-Cafe-1), San Francisco (SF-Cafe-3)

| Ingredient        | Avg Daily Usage | Capacity | Unit  | Notes                    |

| Espresso Shot     | 8               | 50       | shot  | ~6 days at avg usage     |
| Milk              | 1,000           | 5,000    | ml    | 5 L container            |
